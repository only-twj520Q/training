Blob对象解释

> Blob对象表示不可变的类似文件对象的原始数据。Blob表示不一定是JavaScript原生形式的数据。File接口基于Blob，继承了blob的功能并将其扩展使其支持用户系统的文件。
>
> 摘自MDN

我的理解就是Blob对象是表示文件对象的原始数据（二进制）
